# Manual Checklist Verification Workflow

Purpose: Confirm manual verification steps ran successfully and the task card is ready to close.

<L1>
## Core Steps
1) Review the manual verification checklist and ensure all items are complete.
2) Confirm evidence is present (notes, screenshots, or logs).
3) Confirm all task-card steps are complete.
4) Record closeout and move to the next task card step.
</L1>

## Evidence Required
- Use `resources/verification-evidence.md` to confirm proof of manual verification.
- Checklist reference: `resources/task-closeout-checklist.md`.

## If Evidence Is Missing
- Instruct to complete the manual checklist, capture evidence, then return to this step.

## Closeout
- Use `resources/task-closeout-checklist.md`.
