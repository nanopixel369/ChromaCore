# Verification Evidence Guide

Use this reference to confirm the chosen verification method actually ran.

## Acceptable Evidence

- Command output captured in the task log
- Test report files (if generated)
- CI job output (if applicable)
- Local artifacts (coverage files, lint reports)

## Evidence Checklist

- The command matches the required verification type
- The command completed without errors
- Any failure output is resolved and re-run
- Output or artifacts are attached or summarized in the task card
