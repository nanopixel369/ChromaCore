## Evaluation Phase

Perform comprehensive evaluation of the paired OS:

### Security Analysis (5 Layers)

1. **Input Validation**
   - User inputs sanitized?
   - Path traversal protection?
   - Injection prevention?

2. **Execution Control**
   - Commands parameterized?
   - Shell injection risks?
   - Resource limits enforced?

3. **Output Sanitization**
   - Sensitive data exposure?
   - Information leakage?

4. **Privilege Management**
   - Least privilege principle?
   - Unnecessary permissions?

5. **Self-Protection**
   - Skill modification controls?
   - Evolution boundary enforcement?

### Quality Assessment

- **Code Quality:** Clarity, organization, maintainability
- **Documentation:** Completeness, accuracy, examples
- **Structure:** Proper L1/L2 separation, modularity
- **Functionality:** Correctness, edge cases, error handling

### Utility Analysis

- **Problem-Solving Value:** Real impact on task completion
- **Efficiency:** Speed, resource usage
- **Usability:** Clear triggers, intuitive workflow

Generate evaluation report in context.
